Eqol-Infra


Codes phonétiques

Les codes phonétiques correspondent à des caractères standards afin d'assurer la compatibilité entre les différents systèmes. Les correspondances avec les caractères IPA sont données dans le tableau en bas de page.

Deux points particuliers sont à noter :

• Le caractère dièse (#) est aussi utilisé pour définir les associations graphème-phonème afin d'indiquer un graphème silencieux (e. g., le graphème 't' à la fin du mot 'fort'; le graphème 'p' dans le mot 'baptême'; le 'e' toujours muet de 'fée' ou 'année').

• Distinction entre les différents cas de 'e' (muet ou prononcé) :

- Schwa Obligatoire.  En milieu de mots le schwa est obligatoire quand il est précédé par un groupe de consonnes et qu'il est suivi par une consonne (loi dite des 3 consonnes : 'marguerite', 'vendredi'). En début de mots, le schwa est prononcé quand sa suppression conduit à la création d’un groupe de consonnes soit illégal (/Rn/ dans 'renard'; /ls/ dans 'leçon') soit imprononçable (/ss/ dans 'ceci'). Dans ces cas, le 'e' est codé '%'.

- 'e' toujours muet. Le 'e' n’est jamais prononcé lorsqu’il est précédé par une voyelle, que le 'e' soit une marque de flexion (comme dans 'une amie') ou non (comme dans 'la foie', 'la roue', 'la copie', 'la laitue'. Dans ces cas, le 'e' est codé '#'.

- Schwa Optionnel. Le schwa est dit optionnel dans tous les cas ne répondant pas aux deux précédents critères. Le 'e' est alors codé '°'.
