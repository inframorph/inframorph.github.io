Manulex-Morpho


Codes morphologiques

Les caractères 3, 4, 6 et 7 sont utilisés pour indicer les unités morphémiques (les autres chiffres correspondent à des codes «machine» pour des phonèmes, voir « codes phonétiques »). La liste complète des unités morphologiques considérées est fournie dans le tableau en bas de page.

Flexions nominales de genre et de nombre: Le code 3 indice les flexions de genre et de nombre. Par exemple, le mot 'photos' est codé en finale par le morphogramme '3s' auquel correspond le code phonologique '3#' (le caractère '#' est utilisé pour coder un graphème silencieux, voir « code phonétiques »). Par contre, dans le mot 'parfois', le 's' final ne correspond pas à une flexion de nombre et le graphème final est donc simplement codé 's' auquel correspond le caractère '#'.

Flexions verbales monophonémiques : Le code 4 indice les terminaisons verbales. Par exemple, le mot 'aiderai' est codé en finale avec '4ai' auquel correspond le code phonologique '4E'. Dans le mot 'vrai', le graphème 'ai' ne correspond pas à une terminaison verbale et est donc codé simplement 'ai' auquel correspond 'E'. Il est important de rappeler que, dans Manulex-Morpho, les associations G-Ph et Ph-G intégrées pour le codage de la morphologie verbale ne prennent en compte que celles qui sont monophonémiques. Par exemple: '4er' (chant-er), mais pas '4ir' ni '4oir (fin-ir, sav-oir); '4ai' (je chant-ai) mais pas '4erai' (je chant-erai); '4ez' (vous chant-ez) et non '4iez' ou '4issez' (vous chant-iez, vous fin-issez)."

Finales de mots entendues dans les formes fléchies/dérivées (supports de flexion et/ou dérivation) : Le code 6 indice les différentes marques flexionnelles ou dérivationnelles de fin de mot. Par exemple, le 's' final de 'gris' est codé '6s' auquel correspond le code '6#'. Il en est de même pour le 'd' de 'grand' (codé '6d') qui sert d'appui à une forme fléchie (grande) et à une dérivation (grandeur).

Adverbes en <(m)ent> : Le code 7 indice le 'ent' des adverbes de manière, en '-ment'. Par exemple, '7ent' est associé au code phonologique '7@' dans le mot 'simplement'.
